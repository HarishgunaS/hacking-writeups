Leviathan0 Writeup

After sshing into leviathan0, the first thing I did was ls. 

$ ls
The directory appeared to be empty at first, but obviously that couldn't be the case. 
        $ ls -a
        .  ..  .backup  .bash_logout  .bashrc  .profile
Other than the regular hidden files, there was another hidden directory named ".backup". 
        $ cd .backup
        $ ls
        bookmarks.html
        $ cat bookmarks.html
The output of this file was way too long to parse visually. 
        $ cat bookmarks.html | grep pass
        <DT><A HREF="http://leviathan.labs.overthewire.org/passwordus.html | This will be fixed later, the password for leviathan1 is rioGegei8m" ADD_DATE="1155384634" LAST_CHARSET="ISO-8859-1" ID="rdf:#$2wIU71">password to leviathan1</A>
The password of leviathan1 was revealed here as rioGegei8m.
